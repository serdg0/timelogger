# Time Logger
An app for tracking time for multiple activities.


It saves your activities and make a log of time you've spent on each.

Built With:
- Javascript
- React
- React-redux
- React-router
- lodash
- npm
- Ruby on Rails
- CSS 
- Html

Getting Started

Open the [Live Demo Link](https://clock-cms.herokuapp.com/signin) and enjoy.
